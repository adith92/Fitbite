# Fitbite Project Plan

## Product Summary

Fitbite is a healthy food-tech app for Indonesia. It combines body check, nutrition target, pantry ingredients, AI recipes, healthy catering order, admin dashboard, Telegram notification, and business reports.

## MVP Priority

1. Clean repo and deploy base.
2. Customer body check.
3. Pantry ingredient input.
4. Recipe recommendation UI.
5. Catering menu by user goal.
6. Manual order checkout.
7. Admin dashboard.
8. Telegram order notification.
9. Report dashboard.

## User Goals

- Diet / Fat Loss.
- Cutting.
- Bulking.
- Maintain.
- Healthy Daily.

## Customer Features

- Register and login.
- Profile and body data.
- Calorie and macro target.
- Ingredient input.
- Recipe suggestions.
- Nutrition facts.
- Catering order.
- Payment proof upload.
- Order status.

## Admin Features

- Order dashboard.
- Member dashboard.
- Menu management.
- Program management.
- Kitchen queue.
- Delivery queue.
- Payment confirmation.
- Reports.
